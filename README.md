# Online-Food-Ordering-System
## 📌 About the Project
This project is a full-stack Food Delivery Web Application built to understand how real-world platforms like Zomato and Swiggy work.  
It allows users to browse food items, manage cart, and place orders efficiently.

---

## 🚀 Features
- User Authentication (Login / Signup)
- Browse Food Items and Restaurants
- Add to Cart and Manage Orders
- Place Orders
- Responsive Design

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript / React  
- Backend: Node.js, Express.js  
- Database: MongoDB 

---

## 💡 Purpose of the Project
The main objective of this project is to:
- Strengthen full-stack development skills  
- Understand real-world application flow  
- Improve problem-solving and debugging
